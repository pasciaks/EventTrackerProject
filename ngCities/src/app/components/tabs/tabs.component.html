• Grading This is a graded project. You are expected to have your project completed by noon on Monday. You will be given either a pass or fail based on whether your code works given all of the following test conditions: A new event object implements full CRUD. All interactions with the database are done so RESTfully. App uses angular to access data and manipulate the DOM. App presents the aggregated data in some additional format. If the project does work with all of the above test conditions, you will be given a 1 for this week's project. If the project does not work with the above test conditions, you will be given a 0 for this week's project. If you get a zero on the project, you can upgrade to a score of .5 if you turn in a working project by the start of class the following Monday morning AND notify an instructor that you wish to get partial credit. To turn in a project, you must push it to GitHub. You must also deploy it to your AWS server. You must include a README.md that describes your program and how to access it on AWS.
